# Guía Pedagógica Integral - Maestría en Salud Pública

Este repositorio contiene la **Guía Pedagógica Integral para el Proceso de Trabajo de Grado** de la Maestría en Salud Pública de la Fundación Universitaria del Área Andina (Areandina).

## 🚀 Descripción

La Guía Interactiva es una herramienta diseñada para orientar a los estudiantes a través de las diferentes etapas de su formación investigativa y los trámites administrativos necesarios para la culminación de su trabajo de grado.

### Características Principales:

- **Trayectoria de Investigación**: Visualización interactiva de los 4 Seminarios de Investigación.
- **Modalidades de Grado**: Información detallada sobre las opciones de Trabajo de Investigación y Auxiliar de Investigación.
- **Gestión Administrativa**: Flujo paso a paso para la postulación de pre-proyectos y solicitud de avales éticos (CEBIS).
- **Diseño Premium**: Interfaz moderna basada en la identidad visual de Areandina, con diseño responsivo y micro-animaciones.

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica.
- **CSS3**: Diseño personalizado, variables CSS para temas y animaciones fluidas.
- **JavaScript**: Lógica para la navegación entre secciones y el sistema de pestañas.
- **FontAwesome**: Iconografía pedagógica.
- **Google Fonts (Lexend)**: Tipografía moderna y legible.

## 📖 Cómo usar la guía

1. Descarga los archivos de este repositorio.
2. Abre el archivo `guia interactiva.html` en cualquier navegador web moderno (Chrome, Firefox, Safari o Edge).
3. Navega a través de las pestañas principales (**Formación Académica**, **Modalidades de Grado**, **Trámites y Gestión**) para acceder a la información.

## 📂 Estructura del Proyecto

- `guia interactiva.html`: Archivo principal que contiene la estructura, estilos y lógica.
- `Logo_de_Areandina.svg.png`: Recurso visual oficial de la institución.
